# `.claw/design/`

**Design reference material** for the ClawCode project (visual language, UI benchmarks, aesthetics). This is **documentation and inspiration**, not runtime code consumed by the agent at execution time unless explicitly referenced in prompts.

## Subfolders

| Path | Purpose |
|------|---------|
| [`UI/`](./UI/) | Curated **product / brand UI references** (screenshots or captures) used to align TUI/visual work. |
| [`Meta Aesthetics/`](./Meta%20Aesthetics/) | Notes or assets about cross-cutting aesthetic direction. |
